# 竞彩足球智能分析系统

自动分析竞彩足球比赛，生成三票投注策略和HTML报告。

## 使用方法

### 方法1: 双击运行
直接双击 `运行分析.bat` 文件

### 方法2: 命令行运行
```bash
cd 体彩目录
python -m football_analyzer.main
```

## 功能特点

### 数据源
- **竞彩赔率**: 实时抓取竞彩官网 (sporttery.cn) 的赛事和赔率数据
- **天气信息**: 通过 Open-Meteo API 获取比赛城市天气
- **球队数据**: 基于赔率和排名进行多维度分析
- **历史交锋**: 基于排名差距进行近似模拟

### 多维度分析 (9个维度)
1. **赔率隐含概率** (25%) - 从赔率倒推市场共识
2. **近期状态** (20%) - 基于赔率推断近期战绩
3. **历史交锋** (15%) - 排名差距模拟H2H记录
4. **主场优势** (10%) - 主客场战绩拆分
5. **排名实力** (10%) - 联赛排名对比
6. **天气影响** (5%) - 温度、风速、降水
7. **赛程疲劳** (5%) - 比赛密度评估
8. **比赛动机** (10%) - 保级/争冠动机
9. **综合评分** - 加权计算最终推荐

### 三票策略
- **第一票 (保本票)**: 50%资金，2串1，低风险保本
- **第二票 (中赔票)**: 40%资金，3串1，中风险中等回报
- **第三票 (高倍票)**: 10%资金，4-5串1，高风险高回报

### 输出报告
生成精美的HTML报告，包含:
- 所有可购买比赛列表
- 每场比赛的推荐选项
- 三票策略详细配置
- 6种情景的收益分析
- 重要提示和注意事项

## 技术栈
- **后端**: Python 3.12
- **数据**: urllib (无第三方依赖)
- **前端**: 纯HTML/CSS (无JavaScript框架)

## 注意事项
- 竞彩停售时间为23:00，请在停售前完成购买
- 部分比赛可能已开赛，请确认是否仍可购买
- 本系统仅供参考，请理性投注
- 伤缺信息需要人工补充确认
